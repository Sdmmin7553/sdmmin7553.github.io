## ⚙️ How it works

This tool combines many official Windows support utilities into one place. When you run the application, it performs a series of background tasks to verify the health of your operating system.

### System File Checker
The tool runs the `sfc /scannow` command. This scan looks through every protected system file on your computer. If it finds a file that does not match the original version from Microsoft, it replaces that file immediately. This fixes most startup issues and general system instability.

### DISM Utility
The tool uses the Deployment Image Servicing and Management (DISM) utility. This tool repairs the Windows image that SFC uses to fix your files. If your Windows installation is corrupted, DISM restores the components to their default state.

### Runtime Libraries
Many programs rely on specific software components to run. This tool checks for missing Visual C++ Redistributable files and .NET Framework versions. If your computer lacks these, the installer adds them. This action resolves common errors like "MSVCP140.dll missing" or "VCRUNTIME140.dll not found."

### DirectX Repair
If your display flickers or if games refuse to launch, the tool inspects your DirectX drivers. It verifies that your graphics software matches what your games expect.
